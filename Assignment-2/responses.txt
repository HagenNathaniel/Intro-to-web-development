Data mover:
Data Travels through the internet in PACKETS, each packet can contain up to 1,500 Bytes. They will travel from one machine to another 
until they reach their destination.

IP address VS. DNS:

An IP address is used by the DNS to direct a query from it's source, to it's destination. A DNS server stores and directs traffic, much like a
phonebook.

What is a cookie:

A cookie is a small file stored on a users computer, and holds a modest amount of data about that user, and is accessed by either the client, or 
a web service.

URL Parts:

1. Protocol. 2. Server Name. 3. Resource ID.
